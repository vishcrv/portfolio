below is the card that'll appear when your portfolio is shared on LinkedIn, WhatsApp, Discord, etc. screenshot it and save as og-image.png in the repo root.

  1. right-click the card → Inspect
  2. in the Elements panel, click the <div class="og-card"> element
  3. right-click that node → Capture node screenshot (Chrome / Edge / Brave)
  4. browser downloads a pixel-perfect 1200×630 PNG. rename to og-image.png, drop in repo root, commit & push.

alternative — DevTools → device toolbar → set dimensions to 1200 × 630 → screenshot the viewport.
